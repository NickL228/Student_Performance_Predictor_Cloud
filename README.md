# Student Performance Predictor

This project is a full stack web application created for my university coursework.

The application predicts student performance using an AI model. Users can register, log in, make predictions, and view prediction history.

Technologies used:
- React.js
- Django
- MySQL
- Docker
- Nginx

The project was also deployed to a DigitalOcean cloud server.

Basic penetration testing was performed using Nmap.
